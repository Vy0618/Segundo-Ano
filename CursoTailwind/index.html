O QUE É TAILWIND?

Tailwind CSS é um framework de CSS utilitário. Ao invés de classes pré-prontas (como "btn" ou "card"), ele oferece classes atômicas para construir designs diretamente no HTML.

Vantagens

  • • Desenvolvimento mais rápido
  • • CSS consistente
  • • Responsivo fácil
  • • Sem nomes criativos

Desvantagens

  • • HTML poluído
  • • Curva de aprendizado
  • • Configuração inicial

2. INSTALAÇÃO (resumo)

npm install -D tailwindcss@3

Instala o Tailwind

npx tailwindcss init

Cria o config file

No tailwind.config.js:

content: ["./src/**/*.{html,js}"]

No input.css:

@tailwind base;
@tailwind components;
@tailwind utilities;
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch

Gera o CSS

Depois é só linkar o output.css no HTML e instalar a extensão Tailwind CSS IntelliSense no VS Code.
