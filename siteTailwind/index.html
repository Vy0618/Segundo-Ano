O QUE É TAILWIND?

Tailwind CSS é um framework de CSS utilitário. Ao invés de classes pré-prontas (como "btn" ou "card"), ele oferece classes atômicas para construir designs diretamente no HTML.

Vantagens

  • • Desenvolvimento mais rápido
  • • CSS consistente
  • • Responsivo fácil
  • • Sem nomes criativos

Desvantagens

  • • HTML poluído
  • • Curva de aprendizado
  • • Configuração inicial

[ INSTALAÇÃO ]

C:\> No PowerShell:

npm install -D tailwindcss@3
npx tailwindcss init

[OK] Instala o Tailwindcss e cria o config file

C:\> No tailwind.config.js:

content: ["./*.html"]

[INFO] Ele vai procurar por arquivos html por toda a pasta aonde o código está armazenado.

C:\> No root da pasta:

Criar uma pasta "src"
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch

[OK] Gera o arquivo "output.css"

C:\> No input.css:

@tailwind base;
@tailwind components;
@tailwind utilities;

[INFO] Depois é só linkar o output.css no HTML e instalar a extensão Tailwind CSS IntelliSense no VS Code.
